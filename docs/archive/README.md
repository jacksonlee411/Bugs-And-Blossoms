# 文档归档目录说明

本目录用于存放已作废、已被替代、或仅具历史过程价值的仓库级文档。

## 归档规则

- 归档文档不再作为活体 SSOT 引用。
- 归档文档需在标题或状态中明确为历史参考。
- 新增归档时，需同步更新 `AGENTS.md` 与 `docs/dev-plans/013-docs-creation-and-governance-guide.md` 的归档口径。
